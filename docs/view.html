view¶

view.bot¶

Содержит объект бота Telegram.

Инициализирует объект бота с помощью токена, полученного от Telegram, из файла .env Получает токен платёжного провайдера.

view.bot.bot = <telebot.TeleBot object>¶

Объект бота Telegram.

Используется для взаимодействия с API Telegram, отправки сообщений, обработки входящих сообщений и других операций.

Type:

telebot.TeleBot
